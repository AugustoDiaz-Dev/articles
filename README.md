## How to publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload these files.
3. Go to Settings.
4. Go to Pages.
5. Under Build and deployment, choose:

   * Source: Deploy from a branch
   * Branch: main
   * Folder: / root
6. Save.
7. GitHub will generate a public website URL.
